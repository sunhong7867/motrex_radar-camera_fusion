1. Build Package

catkin_make

2. Source Environment
source devel/setup.bash

3. Execution Steps

(A) One-shot Consumer Calibration (Perception + Consumer GUI together)
roslaunch autocal autocal.launch

(B) Developer flow (Perception + real_gui together)
roslaunch autocal autocal_dev.launch

(C) Real sensor topics only launch (Radar + Camera topic publisher)
roslaunch autocal autocal_real_sensors.launch

(C-1) Calibration GUI / pipeline launch (run in another terminal)
roslaunch autocal autocal.launch
or
roslaunch autocal autocal_dev.launch

(D) Topic-only quick check (without calibration pipeline)
roscore

catkin_make --pkg 4fn_basic_viewer
